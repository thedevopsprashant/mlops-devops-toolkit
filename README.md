# mlops-devops-toolkit
MLOps and Devops related scripts, code and steps to leverage AI in Production Projects

# AI-Driven DevOps & MLOps Platform Engineering

This repository demonstrates modern DevOps + MLOps practices using:

- AWS EKS
- Kubernetes
- Terraform
- Docker
- Jenkins
- GitLab CI/CD
- Python Automation
- Prometheus & Grafana
- AI-assisted Operations (AIOps)

## Key Features

### AI-Based Log Analysis
Python-based intelligent log parser to detect anomalies and recurring infrastructure failures.

### Kubernetes Auto-Healing
Automated remediation scripts for unhealthy pods and failed deployments.

### Infrastructure as Code
Terraform modules for provisioning scalable AWS EKS clusters.

### CI/CD for ML Workloads
GitLab and Jenkins pipelines for containerized ML deployments.

### Cost Optimization
Python automation to identify underutilized Kubernetes resources.

### Observability Stack
Prometheus + Grafana integration for production-grade monitoring.
